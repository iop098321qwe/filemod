# Filemod

File utility helpers for archives, hashing, and quick backups.
Wraps common operations with consistent output and options.

## Functions
- `extract`: Extract common archive formats.
- `filehash`: Generate hashes for files with multiple algorithms.
- `backup`: Create a timestamped backup copy of a file.

## Aliases
- `ext`: Alias for `extract`.
- `fh`: Alias for `filehash`.
